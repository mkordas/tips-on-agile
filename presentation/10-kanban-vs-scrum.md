# Kanban vs. Scrum

|            | Kanban            | Scrum            |
|------------|-------------------|------------------|
| Changes    | Not during sprint | Anytime          |
| Roles      | None              | Team/SM/PO       |
| Metrics    | Cycle time        | Velocity         |
| Cadence    | Sprint            | Continuous       |
| Estimation | Optional          | Mandatory        |
| Board      | Persistent        | Resets           |
| New items  | Always            | For new sprint   |
